Python-Pandas語法

pandas中N/A值處理
-

  • 如果要將N/A值補上資料，使用.fillna
    DataFrame.fillna(value=None, method=None, axis=None, inplace=False, limit=None, downcast=None)

  • method : {‘backfill’, ‘bfill’, ‘pad’, ‘ffill’, None}, default None

    pad 或 ffill 上一個值
    backfill 或 bfill 下一個值

  • 如果要填入平均值則
    people[‘age’].fillna(people[‘age’].mean())

  • 如果要刪除存在N/A值的列,使用.dropna
    DataFrame.dropna(axis=0, how=’any’, thresh=None, subset=None, inplace=False)[source]

  • 採用插補法，使用.interpolate
    Series.interpolate(method=’linear’, axis=0, limit=None, inplace=False, limit_direction=’forward’, downcast=None)
    method : {‘linear’, ‘time’, ‘index’, ‘values’, ‘nearest’, ‘zero’,
    ‘slinear’, ‘quadratic’, ‘cubic’, ‘barycentric’, ‘krogh’, ‘polynomial’, ‘spline’, ‘piecewise_polynomial’, ‘from_derivatives’, ‘pchip’, ‘akima’}

    Please note that only method=’linear’ is supported for DataFrames/Series with a MultiIndex

  • 將文字轉為數值格式，使用.to_numeric
    pandas.to_numeric(arg, errors=’raise’, downcast=None)

  • 移動平均
    .rolling
    DataFrame.rolling(window, min_periods=None, freq=None, center=False, win_type=None, on=None, axis=0, closed=None)
    ex. 三日移動平均stock['Close'].rolling(window=3).mean()

QGIS

地圖視覺化

6月10日參加地圖視覺化分享與實作課程，學習QGIS軟體使用，
課堂上練習使用內政部出生登記子女從姓資料，分析台灣各縣字在從母姓比例上是否有所差異。
從圖中可看出，東部及南投地區從母性比重較高，或許和原住民文化相關。

課程結束後，嘗試將各縣市鄉鎮市區的人口數進行漸層色塊呈現，並將雄獅旅行社的門市據點位址呈現在地圖上。
處理步驟包含：

  • Python爬蟲雄獅旅行社門市據點資料
  • 使用Python透過Google MAP API將地址轉換取得經緯度
  • 自內政部取得鄉鎮市區人口統計資料
  • 透過QGIS導入縣市界圖、鄉鎮市區界圖、人口數及雄獅門市經緯度資料進行套圖呈現

一開始找不到經緯度資料導入的地方，因為csv檔導入我選擇＂加入向量圖層＂，正確應選＂加入分隔文字圖層＂

  • 細看雄獅門市地點主要都在人口數較多的地方
  • 台北市就萬華沒有門市

試著下載”105年12月臺北市統計區人口統計_最小統計區”資料來看，想計算門市方圓XX公尺的人數。查詢結果，可以使用buffer功能，實際運作待實作。

註：Google Drive預設圖片網址，無法正常呈現出來，經google後將 ? 前的open改為uc即可正確呈現圖片
